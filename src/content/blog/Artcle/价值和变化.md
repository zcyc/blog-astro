| 类型     | 举例                                 | 价值 |
| -------- | ------------------------------------ | ---- |
| 生活能力 | 沟通、演讲、学习、Office、开车、做饭 | 1    |
| 管理能力 | 组织、协调、分配、推进               | 2    |
| 架构能力 | 方法论、架构设计                     | 3    |
| 抽象能力 | 抽象、总结、算法                     | 4    |
| 工作能力 | 语言、业务、中间件                   | 5    |
| 特长能力 | 艺术、业余爱好                       | 6    |

# 变

语言和框架是变化的，变化频繁的，不稳定的，适当了解，提高工作效率。

# 不变

思想和架构是不变的，变化较少的，比较稳定的，深究原理，形成自己的方法论。
